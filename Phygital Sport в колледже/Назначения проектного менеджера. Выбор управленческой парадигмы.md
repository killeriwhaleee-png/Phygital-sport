### Требования к проектному менеджеру

Проектный менеджер (ПМ) должен сочетать общие профессиональные компетенции со знанием специфики фиджитал-спорта и образовательного процесса.

| Область знаний                            | Что конкретно должен знать и уметь ПМ                                                                                                                                                                                                                                                                                             |
| ----------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Общепрофессиональные знания**           | • Основные методологии управления проектами.  <br>• Навыки планирования, бюджетирования и тайм-менеджмента.  <br>• Умение работать с документацией, оценивать риски, управлять командой и коммуникациями.                                                                                                                         |
| **Предметная область (Phygital Sport)**   | • Концепцию фиджитал-спорта как гибрида физической активности и цифровых технологий.  <br>• Технологические аспекты (VR, игровые платформы, спортивный инвентарь).  <br>• Официальные документы (например, Концепцию развития фиджитал-движения до 2030г<br>• Практику интеграции в учебный процесс (опыт медицинского колледжа). |
| **Специфика образовательного учреждения** | • Особенности планирования и финансирования в колледже.  <br>• Процесс согласования инициатив с администрацией.  <br>• Методы вовлечения студентов (через учебные проекты, конкурсы, как «IT-Планета»).                                                                                                                           |

### Сравнение управленческих парадигм


| Параметр                   | **Scrum** (жёсткая структура)                  | **Kanban** (гибкий поток)                                                         | **Решения для проекта в колледже**                                                                        |
| -------------------------- | ---------------------------------------------- | --------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------- |
| **Структура работы**       | Чёткие итерации (спринты 1-4 недели).          | Непрерывный поток задач без жёстких сроков.                                       | **Scrum** для основных этапов (подготовка к турниру, внедрение модуля).                                   |
| **Роли в команде**         | Строгие роли: Scrum Master, Владелец продукта. | Нет предопределённых ролей.                                                       | Скорее подходит **Scrum**, так как роли преподавателей, тренеров, техников могут быть четко распределены. |
| **Гибкость к изменениям**  | Изменения не приветствуются в течение спринта. | Изменения можно вносить в любое время.                                            | **Kanban** для текущей поддержки, обработки срочных заявок (ремонт оборудования).                         |
| **Визуализация**           | Используется Scrum-доска для спринта.          | Используется Kanban-доска с колонками (например, «To Do», «In Progress», «Done»). | **Kanban** нагляден для отслеживания множества мелких задач.                                              |
| **Измеримость результата** | Измеряется скорость команды (velocity).        | Измеряется время выполнения задач (cycle time).                                   | **Scrum** для контроля достижения ключевых целей (например, к определённой дате).                         |

**Вывод:** Для проекта Phygital Sport в колледже с его разнородными задачами лучше всего подойдет **гибридная модель (Scrumban)
