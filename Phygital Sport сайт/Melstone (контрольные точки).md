| **1. Проектирование и запуск MVP (0-4 месяца)**     | **1.1. Утвержденная концепция и прототип**              | 1-й месяц     | • Определить цели, целевую аудиторию и ключевые требования к сайту (см. файл «Основные требования»).  <br>• Сформировать команду (менеджер, дизайнер, контент-специалист).  <br>• Создать и утвердить прототип (wireframe) и дизайн-макет основных страниц. |
|                                                     | **1.2. Запущенный MVP-сайт**                            | 3-4-й месяц   | • Зарегистрировать домен, выбрать хостинг.  <br>• Разработать и разместить основную структуру: главная, о проекте, новости, контакты, форма обратной связи.  <br>• Наполнить сайт стартовым контентом. Провести тестирование и официальный запуск.          |
| **2. Активное внедрение и развитие (4-10 месяцев)** | **2.1. Внедрены базовые интерактивные функции**         | 6-й месяц     | • Запуск системы онлайн-регистрации на мероприятия.  <br>• Внедрение интерактивного календаря событий.  <br>• Начало ведения регулярной новостной ленты (не реже 1 раза в неделю).                                                                          |
|                                                     | **2.2. Сформировано активное сообщество пользователей** | 9-10-й месяц  | • Количество зарегистрированных пользователей на сайте достигло 100+.  <br>• >70% регистраций на мероприятия проходят через сайт.  <br>• Запущены и работают личные кабинеты для участников с отображением активности.                                      |
| **3. Масштабирование и интеграция (10-24 месяца)**  | **3.1. Сайт как центр управления проектом**             | 12-15-й месяц | • Интеграция модуля турнирных сеток и результатов.  <br>• Налажена автоматическая синхронизация с социальными сетями проекта.  <br>• Сайт используется для всех внутренних коммуникаций по проекту.                                                         |
|                                                     | **3.2. Запуск монетизации и партнерских программ**      | 18-й месяц    | • Открыт онлайн-магазин мерча или раздел с премиум-контентом.  <br>• Заключены первые контракты на размещение рекламы/спонсорства на сайте.  <br>• Доходы от монетизации сайта покрывают его операционные расходы.                                          |
|                                                     | **3.3. Сайт как масштабируемая платформа**              | 20-24-й месяц | • Архитектура сайта подготовлена для SaaS-модели (возможность белых label-решений).  <br>• Разработана и утверждена документация для передачи опыта (франчайзинг).  <br>• Сайт интегрирован с 1-2 внутренними системами колледжа.                           |
